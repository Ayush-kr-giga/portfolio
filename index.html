My Projects

Student management system

A responsive web application designed to streamline student-faculty interaction. The system features two distinct dashboards — one for students and one for faculty. Faculty members can view a list of students and assign individual tasks, which are instantly reflected on the respective student’s dashboard. Built using React and Tailwind CSS, the application utilizes Context API and Local Storage to manage and persist dummy data, ensuring a smooth and dynamic user experience without the need for a backend. This project showcases efficient state management, UI responsiveness, and component-based architecture.

Healthify.AI

Healthify.AI is a React and Tailwind CSS-based web application that provides users with personalized diet plans and health tips using generative AI (Gemini). Users input their basic health metrics—including age, height, weight, BMI, dietary preferences, and health concerns—and the app intelligently crafts a prompt which is sent to Gemini to generate tailored diet recommendations.

Noise Pollution Detector

A browser-based real-time noise pollution detection tool that captures audio from the user's environment, analyzes its frequency data using the Web Audio API, and visually represents the noise levels on an HTML5 canvas. The visualization changes color from green to red based on loudness, and the app also calculates and displays the current decibel (dB) level. If the noise exceeds a set threshold, the system triggers a warning alert.
